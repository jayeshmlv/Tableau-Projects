# HR Analytics Dashboard | Tableau

## Overview

An interactive Tableau dashboard designed to analyze workforce demographics, employee attrition, job satisfaction, and overall HR performance, helping organizations make data-driven workforce decisions.

## Key Features

- Employee Demographics Analysis
- Attrition Tracking & Insights
- Department-wise Workforce Analysis
- Job Satisfaction Monitoring
- HR KPI Tracking
- Interactive Filters

## Tools Used

- Tableau
- Microsoft Excel
